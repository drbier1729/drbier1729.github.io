Lighting

This scene implements Image-Based Lighting from an HDR skydome, as well as a moveable point light. Both utilize GGX BRDF lighting.

Shadows + Reflections

Shadows from the point light source are rendered using a simple depth-testing algorithm. Reflections are rendered using a dual paraboloid map.

Textures

Textures are read in from file and used to provide normal maps for the floor, walls, water, and teapot. Additionally, procedural textures are used for some of the framed photos hanging on the walls.
